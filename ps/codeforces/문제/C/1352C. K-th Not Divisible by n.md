두숫자 사이에 배수가 몇개 잇는지 아는 함수를 만들면 될듯?

```cpp title:k_th.cpp
#include <iostream>
using namespace std;

int bae(int a, int b, int n){
    return b/n - (a-1)/n;
}

int main(){
    int t;
    int n,k;
    cin>>t;
    int result123 = 0;

    while(t--){
        long long n, k;
        cin >> n >> k;
        long long result123 = k;

        long long q = bae(1, k, n);
        result123 += q;

        while(q > 0){
            q = bae(result123-q+1, result123, n); 
            result123 += q;
        }

        cout << result123 << "\n";
    }

    return 0;
}
```

이런 느낌

bae 함수는
1부터 b까지의 배수의 개수 b/n에서
1부터 a-1까지의 배수(a가 n의 배수일수있으니까)를 빼면됨


```cpp title:bae.cpp
int bae(int a,in b, int n){
	return b/n - a/n+1;
}
```

### git diff
커밋 전에 기존 커밋과 바뀐점을 보기 위해선 diff를 이용한다.

```bash
git diff
```

`j, k` 키로 스크롤 동작

하지만 git diff 는 터미널을 쓸 때 주로 사용하고 vscode를 사용한다면 git difftool을 사용하는 것을 추천함

### git difftool
```bash
git difftool
```

vim 에디터임 
h, j, k, l로 이동

```bash
git difftool
```

- 커밋해시 확인
```bash
git log --online 
```



### 다양한 비교 방식
아래서는 git difftool로 설명하지만 대부분 git diff로도 동작함

- 현재 변경사항과 특정 커밋 비교
```bash
git difftool <커밋해시>
```


- 커밋 간 비교
```bash
git difftool <커밋해시1> <커밋해시2>
```


- 스테이징된 것과 최근 커밋 비교
```bash
git difftool --staged
```


- 특정 파일 하나만
```bash
git difftool 파일명.확장자
```

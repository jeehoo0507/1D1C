옵시디언을 쓰다보면 .obsidian/ 혹은
맥을 쓰다보면 .DS_Store 파일이 항상 포함되는 것을 볼 수 있다.

이는 너무 거슬리기에 이를 로컬에서는 들어가게 하되 push 할 때에는 제외시키는 방법이 있다.


- gitignore 파일에 추가
```bash
echo ".DS_Store" >> .gitignore
```

- 캐시 제거(이미 push 되었을 경우에만)
```bash
git ls-files -z '*파일명' | xargs -0 git rm --cached
```

이후 
```bash
git add . 
git commit -m "커밋"
git push
```
하면 됨

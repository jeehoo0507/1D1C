다중 선형 회귀 분석

x가 1개인 선형 회귀를 ==단순 선형 회귀(Simple Linear Regression)==이라고 하고
다수의 로부터 를 예측하는 회귀를  ==다중 선형 회귀(Multivariable Linear Regression)== 라고 한다.


독립 변수가 많음
![[image-33.png|381x49]]



우리는 위의 식을 벡터의 ==내적==을 통해 나타낼 수 있다.


```
H(x)=XW 
```

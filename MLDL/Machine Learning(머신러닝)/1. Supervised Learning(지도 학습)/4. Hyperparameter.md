하이퍼파라미터

정의
> 머신러닝 모델 학습 전에 ==**사람이 직접 설정**하는 값==

ex)

| 하이퍼파라미터                 | 설명                       |
| ----------------------- | ------------------------ |
| **학습률 (Learning Rate)** | 가중치 업데이트 시 기울기에 곱해지는 계수  |
| **배치 크기 (Batch Size)**  | 한 번의 가중치 업데이트에 사용되는 샘플 수 |
| **에포크 수 (Epochs)**      | 전체 데이터셋을 반복 학습하는 횟수      |
| **옵티마이저 (Optimizer)**   | 경사 하강법의 구체적인 알고리즘        |

- 차이점
모델이 스스로 배우는 것(파라미터)과 
개발자가 모델을 가르치는 방식(하이퍼파라미터)을 설정하는 것의 차이

![[image-15.png|286x277]]

```
머신 러닝에서는 데이터를 셀 수 있는 단위로 구분할 때,

```




2. 가중치와 편향 행렬의 크기 결정

- 두 행렬의 곱 J × K이 성립되기 위해서는 행렬 J의 열의 개수와 행렬 K의 행의 개수는 같아야 한다.
- 두 행렬의 곱 J × K의 결과로 나온 행렬 JK의 크기는 J의 행의 개수와 K의 열의 개수를 가진다.

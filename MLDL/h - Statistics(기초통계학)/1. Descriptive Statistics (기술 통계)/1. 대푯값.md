Measures of Central Tendency

1. ==평균 (Mean)==
```
모든 값을 더해서 개수로 나눈 값
```

$$
\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i
$$
정규분포, 이상치 X


2. ==중앙값 (Median)==
```
데이터를 정렬했을 때 정중앙에 위치한 값
```

- 2n개: 가운데 두 값의 평균
- 2n+1개: 가운데 값

이상치 존재, 왜곡된 분포


3. ==최빈값(Mode)==
```
가장 자주 나타나는 값
```


범주형 데이터(관측값을 ==명확한 그룹이나 범주로 분류==)

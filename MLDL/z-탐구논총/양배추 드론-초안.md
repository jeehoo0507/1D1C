AI 기반 농작물 정밀 탐색 시스템 (ResNet-18)

### 1. 사용 모델: ResNet-18 (Pre-trained)

이 시스템은 전 세계적으로 검증된 **ResNet-18**이라는 인공지능 모델을 사용합니다.

- 특징 추출: 양배추 및 농작물 특유의 패턴을 512차원의 고유한 벡터(Vector)로 변환합니다.

---

### 2. 작동 방식: 멀티 스케일 슬라이딩 윈도우

이 알고리즘은 사람이 돋보기를 들고 밭을 촘촘히 검사하는 과정을 자동화했습니다.

1. 기준 벡터 생성: 사용자가 제공한 '샘플 양배추' 사진에서 AI가 고유한 특징(feature)을 추출하여 기억합니다.
2. 다중 크기 스캔 (Multi-scale):밭 전체 사진 위에서 슬라이딩 방식으로 가장 비슷한 부분을 크롭 사이즈별로 찾습니다
상당히 무식해보일수 있는 방법이지만 목적이 '밭'에서의 세그먼트임으로 sam이나 dino-cnn과 같은 무거운 기술은 오히려 손해라고 생각합니다
3. 정밀 비교: 모든 조각을 AI 모델에 넣어 기준 양배추와 얼마나 닮았는지 유사도 점수를 매깁니다. 단순 밝기 차이가 아닌 '형태의 유사성'을 보기 때문에 흰 바탕 같은 노이즈를 효과적으로 걸러냅니다.
4. 중복 제거 (NMS): 한 양배추 위에 여러 개의 상자가 겹쳐서 발견되면, 가장 정확도가 높은 상자 하나만 남기고 나머지는 자동으로 정리합니다.

$$s_i = \begin{cases} s_i, & \text{if } \text{IoU}(M, B_i) < N_t \ 0, & \text{if } \text{IoU}(M, B_i) \ge N_t \end{cases}$$

---

### 3. 시스템의 강점

- 정확성: 랜덤 가중치가 아닌, 학습된 AI의 판단력을 빌려 쓰므로 인식 오류가 매우 적습니다.
- 유연성:별도의 추가 학습 없이도 기준 사진(양배추, 무, 감자 등)만 바꿔주면 다양한 작물에 즉시 적용 가능합니다.
- 신뢰도: 흰색 비닐이나 흙 같은 배경 요소를 형태적 분석을 통해 명확히 구분해냅니다.
- 절대적 시간:같은 해상도 한에서는 굉장히 오차율이 적은 즉 케이스 별 테스트 시간이 같습니다

$$\theta_t \leftarrow \lambda \theta_t + (1 - \lambda) \theta_s$$

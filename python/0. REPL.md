콘솔 화면에서 파이썬 구문을 입력하면 바로 결과를 반환하고 다시 입력할 수 있는 도구
**R**ead **E**valuate **P**rint **L**oop



Mac, 리눅스, 윈도우에서 `python`, `python3` 
리눅스에서는 버전명을 치거나 따로 설정해야함


### 사용법
상당히 재밌는 사용법들이 다수 존재한다.

1. 변수
```bash
>>> x = 5        # 대입문(statement) → 화면에 아무것도 안 뜸
>>> x * 10       # 표현식(expression) → 결과가 자동으로 출력됨
50
```

2. 값 재사용
```bash
>>> y = 5
>>> y * 2
10
>>> _ + 3
13
```
`_`을 통해 마지막 출력값을 재사용 가능하다.

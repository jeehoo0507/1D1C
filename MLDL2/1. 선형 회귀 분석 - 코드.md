### 4) 가설(model 설정)
모델 선언은 low-level, nn.Module, class 를 이용한 방법이 있다.

$$H(x) = Wx + b$$


- Low-level
```python
W = torch.zeros(1, requires_grad=True)
b = torch.zeros(1, requires_grad=True)
optimizer = optim.SGD([W, b], lr=0.01)

# 가설
hypothesis = x_train * W + b
```

- nn.Module
```python
model = nn.Linear(1, 1)
optimizer = optim.SGD(model.parameters(), lr=0.01)

# 가설
hypothesis = model(x_train)
```

- class
```python
class LinearRegressionModel(nn.Module):
    def __init__(self):
        super().__init__()
        self.linear = nn.Linear(1, 1)
    def forward(self, x):
        return self.linear(x)

model = LinearRegressionModel()
optimizer = optim.SGD(model.parameters(), lr=0.01)

# 가설
hypothesis = model(x_train)
```

### 5) 손실 함수 
이 또한 low-level과 nn.functional을 이용한 방법이 존재한다.
$$cost(W, b) = \frac{1}{n} \sum_{i=1}^{n} \left[ y^{(i)} - H(x^{(i)}) \right]^2$$
- low-level
```python
cost = torch.mean((hypothesis - y_train) ** 2)
```

- nn.functional
```python
cost = F.mse_loss(hypothesis, y_train)
```


### 6) 경사하강법
$$W := W - \alpha \frac {\partial cost(W,b)}{\partial W}$$
$$W := W - \alpha \frac {\partial cost(W, b)}{\partial b}$$
- low-level
```python
optimizer = optim.SGD([W, b], lr=0.01)
```

- nn.module && class
```python
optimizer = optim.SGD(model.parameters(), lr=0.01)
```


#### 경사하강법 진행
```python
optimizer.zero_grad()   # 1. 기울기 초기화
cost.backward()         # 2. 역전파로 기울기 계산
optimizer.step()        # 3. 기울기 방향으로 파라미터 업데이트
```


### 7) 전체 코드

- low-level
```python
import torch
import torch.nn as nn
import torch.nn.functional as F
import torch.optim as optim

x = torch.FloatTensor([[1],[2],[3]])
y = torch.FloatTensor([[2],[4],[6]])

W = torch.zeros(1, requires_grad=True)
b = torch.zeros(1, requires_grad=True)

optimizer = optim.SGD([W, b], lr=0.01)

nb_epoch = 1000
for epoch in range(1, nb_epoch + 1):
    hypothesis = W * x + b
    cost = torch.mean((hypothesis - y) ** 2)

    optimizer.zero_grad()
    cost.backward()
    optimizer.step()

    if epoch % 100 == 0:
        print(f"epoch {epoch:4d}/{nb_epoch} W: {W.item():.3f} b: {b.item():.3f} cost: {cost.item():.6f}")

```

- nn.linear
```python
import torch
import torch.nn as nn
import torch.nn.functional as F
import torch.optim as optim

x = torch.FloatTensor([[1],[2],[3]])
y = torch.FloatTensor([[2],[4],[6]])

model = nn.Linear(1, 1)

optimizer = optim.SGD(model.parameters(), lr=0.01)

nb_epoch = 1000
for epoch in range(1, nb_epoch + 1):
    hypothesis = model(x)
    cost = F.mse_loss(hypothesis, y)

    optimizer.zero_grad()
    cost.backward()
    optimizer.step()

    if epoch % 100 == 0:
        params = list(model.parameters())
        W, b = params[0], params[1]
        print(f"epoch {epoch:4d}/{nb_epoch} W: {W.item():.3f} b: {b.item():.3f} cost: {cost.item():.6f}")
```

- class

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
import torch.optim as optim

x = torch.FloatTensor([[1],[2],[3]])
y = torch.FloatTensor([[2],[4],[6]])

class LRmodel(nn.Module):
    def __init__(self):
        super().__init__()
        self.linear = nn.Linear(1, 1)

    def forward(self, x):
        return self.linear(x)

model = LRmodel()

optimizer = optim.SGD(model.parameters(), lr=0.01)

nb_epoch = 1000
for epoch in range(1, nb_epoch + 1):
    hypothesis = model(x)
    cost = F.mse_loss(hypothesis, y)

    optimizer.zero_grad()
    cost.backward()
    optimizer.step()

    if epoch % 100 == 0:
        params = list(model.parameters())
        W, b = params[0], params[1]
        print(f"epoch {epoch:4d}/{nb_epoch} W: {W.item():.3f} b: {b.item():.3f} cost: {cost.item():.6f}")
```

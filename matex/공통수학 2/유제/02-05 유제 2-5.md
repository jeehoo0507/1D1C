# 유제 2-5

## 문제

다음을 만족시키는 직선의 방정식을 구하시오.

1. 직선 $y=\dfrac23x+4$에 수직이고, 점 $(3,-2)$를 지난다.
2. 직선 $3x+2y=3$에 평행하고, 점 $(2,3)$을 지난다.
3. 두 점 $A(-2,-1)$, $B(1,2)$를 지나는 직선에 수직이고, 점 $B$를 지난다.
4. 두 점 $A(2,3)$, $B(5,0)$을 잇는 선분을 수직이등분한다.

## 정답

1. $y=-\dfrac32x+\dfrac52$  
2. $y=-\dfrac32x+6$  
3. $y=-x+3$  
4. $y=x-2$

# S 보기 3

## 문제

다음 집합을 원소나열법으로 나타내시오.

1. $\{x\mid x\text{는 }20\text{보다 작은 소수}\}$
2. $\{x\mid x\text{는 }100\text{보다 작은 자연수 중 }3\text{의 배수}\}$
3. $\{x\mid x=2n+1,\ n\text{은 정수}\}$
4. $\left\{x\mid x=\dfrac1n,\ n\text{은 자연수}\right\}$

## 정답

1. $\{2,3,5,7,11,13,17,19\}$
2. $\{3,6,9,12,\cdots,99\}$
3. $\{\cdots,-3,-1,1,3,5,\cdots\}$
4. $\left\{1,\dfrac12,\dfrac13,\dfrac14,\cdots\right\}$

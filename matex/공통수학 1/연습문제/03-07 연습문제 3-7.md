# 연습문제 3-7

## 문제

$f(x)=2^x(ax^2+bx+c)$가 모든 실수 $x$에 대하여

$$f(x+1)-f(x)=2^x x^2$$

을 만족시킬 때, 상수 $a,b,c$의 값을 구하시오.

## 정답

$$a=1,\quad b=-4,\quad c=6$$

## 풀이

$$f(x+1)=2^{x+1}\{a(x+1)^2+b(x+1)+c\}=2\cdot2^x\{a(x+1)^2+b(x+1)+c\}$$

이므로 $f(x+1)-f(x)=2^xx^2$의 양변을 $2^x$로 나누면

$$2\{a(x+1)^2+b(x+1)+c\}-(ax^2+bx+c)=x^2$$

좌변을 전개하면

$$2(ax^2+2ax+a+bx+b+c)-(ax^2+bx+c)=ax^2+(4a+b)x+(2a+2b+c)$$

이 식이 모든 $x$에 대하여 $x^2$와 같으므로

$$a=1,\qquad 4a+b=0,\qquad 2a+2b+c=0$$

$a=1$에서 $b=-4a=-4$, $c=-2a-2b=-2+8=6$.

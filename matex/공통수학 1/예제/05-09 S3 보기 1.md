# S3 보기 1

## 문제

다음을 간단히 하시오. 단, $x,y$는 실수이다.

1. $$\sqrt{(3-\sqrt{11})^2}$$
2. $$\sqrt{(x^2-2x+4)^2}$$
3. $$\sqrt{x^2-2xy+y^2}$$
4. $x\ge2$일 때, $$\sqrt{(x-2)^2}+|2-x|$$

## 정답

1. $$\sqrt{11}-3$$
2. $$x^2-2x+4$$
3. $$x\ge y\text{일 때 }x-y,\quad x<y\text{일 때 }y-x$$
4. $$2x-4$$

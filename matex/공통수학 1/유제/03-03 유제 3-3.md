# 유제 3-3

## 문제

다음 등식이 $x,y$에 관한 항등식일 때, 상수 $a,b,c$의 값을 구하시오.

$$(a+b)x+(b-2c)y=(c-2)(x-1)$$

## 정답

$$a=-4, b=4, c=2$$

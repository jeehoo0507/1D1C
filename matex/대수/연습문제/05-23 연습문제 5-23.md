# 연습문제 5-23

## 문제

다음 $x$에 관한 두 이차방정식이 공통근을 가지도록 하는 상수 $a$의 값을 구하시오. 단, $a \neq 1$이다.
$$x^2 + x \log 2a + \log(a+1) = 0$$
$$x^2 + x \log(a+1) + \log 2a = 0$$

# 연습문제 16-23

## 문제

이상인 모든 자연수 $n$에 대하여 부등식이 성립함이
$$1 + \frac{1}{2^2} + \frac{1}{3^2} + \cdots + \frac{1}{n^2} < 2 - \frac{1}{n}$$

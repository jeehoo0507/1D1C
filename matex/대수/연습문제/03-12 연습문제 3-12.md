# 연습문제 3-12

## 문제

양수 $A$에 대하여 $\log A$의 소수부분을 $f(A)$라고 하자. 또, $1 \le x < 100$일 때 자연수 $n$에 대하여 $f(x^n) + 2f(x) = 1$을 만족시키는 $x$의 개수를 $g(n)$이라고 하자. $g(5) + g(6)$의 값을 구하시오.

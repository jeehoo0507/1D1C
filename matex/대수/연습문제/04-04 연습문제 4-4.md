# 연습문제 4-4

## 문제

두 지수함수 $f(x) = a^{bx-1}$과 $g(x) = a^{cx}$의 그래프는 직선 $x=2$에 대하여 서로 대칭이다. $f(4) + g(4) = \frac{5}{2}$일 때, $a+b+c$의 값을 구하시오. 단, $a, b, c$는 상수이고, $0 < a < 1$이다.

```python
DataLoader(dataset, batch_size=..., shuffle=..., num_workers=...)
```

dataset만 필수, shuffle은 생략하면 false랑 같음

#### 왜 쓰는가?
dataset 자체로도 가능하지만 병렬 처리를 하기 위함이다. 
결국 대부분의 gpu로 작동시키는 모델들은 행렵곱을 이용할텐데

하나의 이미지를 길게 풀어 하나의 행벡터로 만들면
```
입력          가중치           출력
[N, 784]  ×  [784, 128]  =  [N, 128]
```

이런게 가능하다.

```
[32, 784] × [784, 128] = [32, 128]   ← 32장
[ 4, 784] × [784, 128] = [ 4, 128]   ← 4장, 문제없음
[ 1, 784] × [784, 128] = [ 1, 128]   ← 1장도 가능
```

이처럼 결과 비교는 행끼리만 하면 되기에 로스 계산도 수월하고 모델 학습이 끝나고 추론할때에도 가능하게 동작 할 수 있다.

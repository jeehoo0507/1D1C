
### 기본 문법
```bash
mv 원본경로 목적지경로
```
이것도 폴더 자체를 목적지 폴더에 넣는 것임

- 폴더 자체를 통째로 옮기기 (이름 유지)
```bash
mv 폴더명 옮길위치/
```

예: `mv dataset /Users/choejihu/Desktop/` → Desktop 밑에 `dataset` 폴더가 그대로 생김
바로 상위 폴더 터미널에서만 가능


- 폴더 내부 파일들만 다른 폴더로 옮기기 (폴더 자체는 안 옮김)
```bash
mv 폴더명/* 목적지폴더/
```
`*`가 핵심이에요. 이게 있어야 "폴더 안의 내용물"을 의미하고, 없으면 "폴더 자체"를 옮기려는 뜻이 됩니다.
